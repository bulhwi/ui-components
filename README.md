# 사내UI킷 (MBSW)

React와 Next.js, TypeScript 기반의 사내 전용 UI 컴포넌트 라이브러리

## 개요
일관된 디자인과 빠른 개발을 지원하며 Storybook과 다크모드, 국제화, 반응형 기능을 기본 제공합니다.

## 개발 환경
- React 18
- Next.js
- TypeScript
- Storybook 7+
- styled-components
- Jest + React Testing Library

## 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# Storybook 실행
npm run storybook
```

## 구조
- `/src/components` - UI 컴포넌트
- `/src/stories` - Storybook 스토리
- `/src/styles` - 테마 및 스타일
- `/src/utils` - 유틸리티 함수
